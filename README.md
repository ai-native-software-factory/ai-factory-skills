# ai-factory-skills

存放 AI Factory 所有角色的 Skills 文档。

## 已包含 Skills

- **`brainstorming`** — Superpowers 风格头脑风暴流程。在任何创意工作之前使用，将想法通过协作对话转化为完整设计和规格文档。

## 贡献指南

每个 Skill 为独立目录，包含：

- `SKILL.md` — Skill 主文件（YAML frontmatter + Markdown 正文）
- `references/` — 参考文档（可选）
- `scripts/` — 自动化脚本（可选）

详见 [ai-factory](https://github.com/ai-native-software-factory/ai-factory)
